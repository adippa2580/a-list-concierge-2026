# A-List: Luxury Nightlife Intelligence

A-List is a premium, high-end nightlife application designed for elite discovery and seamless booking. Featuring a location-aware AI Concierge, synchronized artist rosters, and integrated lifestyle APIs (Spotify, Instagram, Eventbrite), A-List delivers a curated experience for the modern socialite.

## Core Features

- **AI Concierge**: A personalized, location-aware assistant that delivers structured venue and event recommendations with an insider tone.
- **Elite Discovery**: High-fashion editorial interface for exploring venues and talent with a fluid **Vertical Motion Experience**.
- **Vibe Matching**: Integration with Spotify to match users with artists and venues that fit their unique sonic profile.
- **Profile Vault**: Secure profile management with cloud-hosted avatar storage and unified account synchronization.
- **Social Integration**: Instagram OAuth for live social dispatches and "The Scene" activity feed.
- **Premium Bookings**: Streamlined flows for guest lists and bottle service reservations at world-class venues.

## Design Aesthetic

A-List follows a **Minimalist High-Fashion Editorial** aesthetic:
- **Dark Monochromatic Palette**: Deep blacks and high-contrast typography.
- **Luxury Textures**: Custom "Black Marble" and "Liquid Gold" CSS effects.
- **Typography**: A refined mix of *Cormorant Garamond* (Serif) for elegance and *Inter* (Sans) for precision.
- **Grayscale Hover Effects**: Subtle visual transitions that activate on interaction.

## Technical Architecture

- **Frontend**: React with Tailwind CSS v4 and Motion.
- **Backend**: Supabase Edge Functions (Hono) for secure API orchestration, KV Store management, and **Supabase Storage** for user assets.
- **Integrations**:
  - **Supabase Storage**: Secure, cloud-hosted storage for user profile media and elite assets.
  - **Spotify API**: User preference mapping and top artist synchronization.
  - **Eventbrite API**: Real-time nightlife event discovery and live data cross-referencing.
  - **Instagram API**: Visual activity feeds and media synchronization.
  - **OpenAI GPT-4o**: Powering the high-end conversational AI Concierge.

## Getting Started

1. **Connect Spotify**: Unlock "Vibe Matching" by linking your music profile.
2. **Set Location**: A-List automatically detects your city to curate "Elite Picks."
3. **Ask Concierge**: Use the floating Sparkle button to find trending spots tonight.
